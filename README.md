# Telegram Crypto Wallet & Domain Balance Bot

Telegram-бот, который автоматически находит адреса криптовалютных кошельков (EVM или Solana) и доменные имена (ENS / SNS) в сообщениях, и присылает в ответ сводку по их балансам.

## ✨ Особенности
- **Поддержка EVM-сетей**: Проверяет балансы на Ethereum, Binance Smart Chain (BSC), Polygon, Arbitrum и Optimism.
- **Поддержка Solana**.
- **Интеграция с доменными именами**:
  - Разрешает ENS-домены (например, `vitalik.eth`) в EVM-адреса.
  - Разрешает SNS-домены (например, `bonfida.sol`) в Solana-адреса.
- **Асинхронность и скорость**: Запросы к доменным API и RPC-узлам выполняются параллельно с помощью `asyncio` и `httpx`.
- **Работа в группах**: Бот отлично подходит для добавления в групповые чаты.

## 🛠️ Настройка и запуск

### 1. Создание бота в Telegram
1. Откройте [BotFather](https://t.me/BotFather) в Telegram.
2. Отправьте команду `/newbot` и следуйте инструкциям для получения **API Token**.
3. **Важно (для работы в группах)**:
   - Отправьте команду `/setprivacy`.
   - Выберите вашего бота.
   - Нажмите **Disable** (Отключить приватный режим). Это позволит боту читать все сообщения в группе, а не только те, которые начинаются с команды.
   - *Альтернатива*: Сделайте бота администратором группы.

### 2. Настройка окружения
1. Создайте файл `.env` из шаблона `.env.example`:
   ```bash
   cp .env.example .env
   ```
2. Откройте `.env` и вставьте ваш токен:
   ```env
   TELEGRAM_BOT_TOKEN=8636816516:AAESZw...
   ```

### 3. Установка зависимостей
Установите библиотеки с помощью pip:
```bash
pip install -r requirements.txt
```

### 4. Запуск
- Для запуска Telegram-бота:
  ```bash
  python bot.py
  ```
- Для отдельного тестирования разрешения доменов:
  ```bash
  python test_domains.py
  ```

## 🔍 Как это работает?
Бот сканирует сообщения на наличие регулярных выражений:
- **EVM (0x...)**: Любая шестнадцатеричная строка из 40 символов после `0x`.
- **Solana (Base58)**: Адрес длиной от 32 до 44 символов, валидируемый по длине байт при base58-декодировании.
- **ENS (domain.eth)**: Находит любые `.eth` домены и преобразует их через API ENSData.
- **SNS (domain.sol)**: Находит любые `.sol` домены и преобразует их через SDK Proxy Bonfida.
